# InternReady — Internship Application Helper

A free tool that helps students turn their skills and projects into a sharp internship application in seconds.

## What it does

- Enter your skills, projects, and extracurriculars
- Pick the role you're applying for
- Get a tailored "Why hire me" summary, cover note opener, and application bullets — instantly

## Built with

- Next.js 14 (App Router)
- TypeScript
- Claude Sonnet 4.6 via Anthropic API (no key needed in client — handled server-side)

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy on Vercel

1. Push this repo to GitHub (make it public)
2. Go to [vercel.com](https://vercel.com) → Import your repo
3. Deploy — no environment variables needed for the free Anthropic API endpoint
4. Done. Your live URL is ready.
